# main

## 소개

War Archive는 고대부터 현대까지의 전쟁 기록을 카테고리별로 보관하고 탐색할 수 있도록 만드는 아카이브 프로젝트입니다. `main` 브랜치에서는 이 아이디어를 실제 웹 프로젝트로 옮기기 위한 첫 구조가 만들어졌습니다.

프론트엔드는 `front/` 디렉터리를 중심으로 구성되며, 프로젝트 설명은 `README.md`와 `structure.md`에 정리되어 있습니다. 아직 백엔드나 Docker 인프라가 핵심 구조로 들어오기 전이므로, 이 버전은 정적 프론트엔드 중심의 시작점으로 볼 수 있습니다.

## 주요 구성

- `front/`: 웹 페이지와 정적 자산을 담는 프론트엔드 영역
- `README.md`: 프로젝트 개요, 기술 스택, 실행 방법 정리
- `structure.md`: 프로젝트 폴더 구조 설명
- `.gitignore`: 개발 중 생성되는 불필요한 파일 관리
- `.gitattributes`: Git 파일 속성 관리
