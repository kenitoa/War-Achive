# version 1.2

## 프론트엔드 중심으로 단순화한 버전

## 소개

`version 1.2`에서는 프로젝트의 범위가 크게 줄어듭니다. 이전 버전에서 추가되었던 로그인 화면, 인증 관련 JavaScript, API 연결 코드가 삭제되고, 백엔드 서버와 Docker 배포 구성도 제거됩니다.

그 결과 War Archive는 다시 `front/` 중심의 정적 아카이브 사이트가 됩니다. 사용자는 전쟁 관련 콘텐츠를 탐색하고, 카테고리별 페이지와 상세 정보를 확인하는 데 집중할 수 있습니다.

## 주요 변화

- 로그인 관련 프론트엔드 파일 제거
- API 연결용 공통 JavaScript 제거
- Express 백엔드 코드 제거
- Docker Compose, Nginx, 배포/백업 스크립트 제거
- API, 백업, 배포, 구조 문서 제거
- README를 현재 프로젝트 범위에 맞게 재정리

## 주요 구성

- `front/`: 프로젝트의 핵심 프론트엔드 영역
- `README.md`: 단순화된 프로젝트 설명
- `.gitattributes`: Git 파일 속성 관리
