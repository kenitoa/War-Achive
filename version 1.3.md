# version 1.3

## version 1.2와 동일한 릴리스 포인터

`version 1.3`은 현재 Git 기록 기준으로 `version 1.2`와 같은 커밋을 가리키는 버전입니다. 코드 변경이 새로 추가된 릴리스라기보다는, `version 1.2`에서 정리된 정적 프론트엔드 상태를 다시 가리키는 버전 포인터에 가깝습니다.

이 문서는 `version 1.3`의 현재 의미를 명확히 남기기 위한 소개 문서입니다.

## 이 버전의 성격

- `version 1.2`와 코드상 동일한 상태
- 별도 추가 기능이 없는 버전 포인터
- 정적 프론트엔드 중심 구조를 유지하는 버전
- 이후 실제 1.3 변경사항을 쌓기 전의 기준점

## 소개

`version 1.3`은 이름상 다음 버전이지만, 현재 저장소에서는 `version 1.2`와 같은 커밋인 `77d307c`를 기준으로 합니다. 따라서 기능, 파일 구조, 제거된 항목 모두 `version 1.2`와 동일합니다.

현재 상태에서 War Archive는 백엔드와 로그인 기능을 제외하고, `front/`를 중심으로 정적 아카이브 사이트를 제공하는 형태입니다. 전쟁 관련 콘텐츠를 카테고리별로 탐색하고 상세 페이지를 확인하는 경험에 집중합니다.

## 주요 구성

- `front/`: 정적 프론트엔드 사이트
- `README.md`: 프로젝트 소개와 실행 정보
- `.gitattributes`: Git 파일 속성 관리

## version 1.2와의 관계

`origin/version-1.2`와 `origin/version-1.3`은 모두 같은 커밋을 바라봅니다. 그래서 두 버전 사이에는 실제 diff가 없습니다.

다만 문서상으로는 `version 1.3`을 "다음 작업을 시작하기 위한 이름 있는 기준점"으로 볼 수 있습니다. 이후 1.3에서 새로운 화면 개선, 데이터 추가, 스타일 정리 등이 진행되면 이 문서도 함께 갱신하면 됩니다.
